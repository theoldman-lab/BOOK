# 写作方法与注意事项（基于第一章实践）

本文档从第一章的实际写作中提炼可复用的方法与约束，供后续五章参考。

---

## 一、结构约束

### 层级限制

- **仅使用两级标题**：`#`（章标题）和 `##`（节标题）。
- **禁止 `###` 及更深层级**。节内以空行分隔的自然段落推进，不以小标题切分。

### 段落组织

- 每个 `##` 节包含 3–6 个自然段落，段落之间以空行分隔。
- 段落长度不严格均等，核心论证段落长于过渡/收束段落。
- 每一节的末段承担两个功能：收束本节论证 + 引出下一节（或下一章）。

### 节间过渡

- 前一节的最后一句话与后一节的第一句话之间，形成追问与回答、悬置与回应、或提出者与实现者的关系。
- 过渡句不写"接下来我们将看到……""下一章将讨论……"这类预告式语言。改用实质性衔接——让内容本身提出下一段的问题。

---

## 二、叙述视角

### 第三人称全局视角

- **绝对禁止第二人称**："你""读者""我们"均不可出现。
- 当需要描述一个一般化的认知情境时，使用"一个商人""一个人""一个希腊人"或直接以人物为主体。
- 不出现"试想""打开一张地图""注意这里""不妨设想"等命令式或邀请式措辞。

### 叙述者姿态

- 叙述者是中立的史述者与分析者，不是教师、导游或布道者。
- 不评判历史人物的对错（如"它当然是错的"——这是第一章中唯一的例外，针对泰勒斯的"水是本原"这一命题因其显而易见的错误而允许一个简短的定性插入，但应极其克制）。
- 对历史人物的立场给予内部理解式的呈现——先让读者感受其说服力，再揭示其局限性。

---

## 三、论证模式

### 物质事实 → 认知后果

- 全书最核心的写作原则。每一段涉及经济、政治、地理等"硬事实"时，其后必须紧跟至少同等篇幅的"这如何改变了他们想问题的方式"。
- 反面示例（不可写）："海上贸易促进了文化交流。"——这是物质事实的陈述，没有翻译为认知后果。
- 正面示例："商人在异族度量体系中反复接触到同一个量可以被不同约定表述这一事实，*nomos*与*physis*的区分由此在交易实践中生根。"

### 具体先于抽象

- 每一层论证都从一个具体的场景、人物、命题或证明开始，然后上升至哲学含义。
- 音乐发现（弦长整数比 → 可听的秩序）→ 天球和谐（可听的秩序 → 可见的秩序）→ 灵魂净化（外在的秩序 → 内在的秩序）。三个层次从具体到抽象递进。
- √2的证明：完整呈现证明步骤（几何或代数均可，力求简洁清晰），然后逐层展开哲学后果。

### 问题先于回答

- 每一节的叙事节奏是：先让读者感到一个真实的问题存在，然后给出历史上的回答方案。
- 不要在问题尚未被感受到之前就提供答案。这会消解叙事的动力。

---

## 四、关键术语的处理

### 希腊术语的引入格式

首次出现时按以下格式：

```
中文译名（*拉丁转写*，字面义为"……"）
```

示例：`*logos*（逻各斯，以同一个词同时意指话语、理性、比例和规律）`

同一术语再次出现时只写拉丁转写，不再重复解释。

### 关键术语的选择

每节只引入 3–5 个真正必要的术语。术语是论证的工具，不是装饰。如果一个术语的出现不推进论证，就删掉它。

---

## 五、语言风格

### 句式

- 以中等长度的陈述句为主体（15–30字）。
- 关键判断使用短句（10字以内），置于段落中或段末作为落点。
- 少见"不仅……而且……""既……又……"等成对的关联词。多数并列关系用逗号或分号处理。

### 修辞手段

- **对比**：城邦辩论 vs 帝国命令；意见 vs 真理；潜无穷 vs 实无穷；发现 vs 证明。对比是全书最核心的修辞策略——它让哲学立场之间的张力可视化。
- **重复变奏**：一个关键判断在不同章节中以变形的方式重现。"说服取代命令，论证取代暴力"——这种句子在全章中只出现一次，但其精神贯穿始终。
- **克制加粗**：加粗仅用于段落中的核心判断句（每段至多一处），不用于术语、不用于人名、不用于书名。

### 禁用措辞清单

- 说教式："我们必须理解""关键在于""值得注意的是""读者应注意到"
- 预告式："接下来我们将看到""下一章将讨论""如前所述"
- 邀请式："试想""想象一下""打开地图""不妨设想"
- 感叹式："多么……！""何等……！"
- 第一/第二人称："我们""你""读者""笔者"

## 六、各节内部的典型结构

以下是从第一章中提炼出的段落功能单元，可在后续各章中复用：

| 功能单元 | 典型位置 | 作用 |
|----------|----------|------|
| 悬念或追问 | 节首第一句 | 承上：将前一节的落点转化为本节要解决的问题 |
| 场景或案例 | 节中前段 | 以具体人物、事件或证明建立讨论对象 |
| 概念展开 | 节中中段 | 从案例中提取哲学概念，阐释其含义与层次 |
| 历史后果 | 节中后段 | 追溯该概念的后续影响，建立跨章节的连线 |
| 收束与引出 | 节末最后一段 | 总结本节论证 → 暴露悬而未决的问题 → 引出下一节 |

## 七、全章情感节奏控制

每一章需要一个总体的情感弧线。以第一章为例：

```
好奇（为什么是希腊？）
  → 信服（万物皆数的说服力）
    → 不安（几何量的伏笔）
      → 震惊（√2的崩塌）
        → 沉思（四大问题）
```

后续各章在动笔前应预先设定情感线，确保叙事有方向。

## 八、切忌的模式

1. **生平流水账**：不因为提到了一个人物就列举其生卒年月、籍贯、师承、著作清单。只有在推进论证时才给出生平信息。
2. **列清单**：不写"有三点原因如下：第一……第二……第三……"。将多点论证融入叙述的层次推进之中。
3. **预判胜负**：不在一场哲学论争的开端就暗示哪一方更"正确"。让双方的观点先以各自最强的版本被呈现。
4. **直接引入现代术语**：不在讨论毕达哥拉斯时使用"形式主义"；不在讨论亚里士多德时使用"经验主义"——除非在整章的收束或遗产总结部分做跨时代的对照。
5. **用表格替代叙述**：表格可以作为辅助呈现对照关系（如第一章的三重冲击），但不能取代论证性的叙述。表格之后必须有文字的展开。
6. **同一段内跳跃跨度过大**：从毕达哥拉斯到康托尔的一千年跨度不能在同一个段落内完成。时间跳跃需要段落分隔或明确的过渡措辞。
